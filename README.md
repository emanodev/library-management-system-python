# Library Management System (Python)

Simple CLI simulation of a library management system built with Python using Object-Oriented Programming.

The system stores all data in memory using lists, without using a database.

## Features

- Register books
- Register users
- Borrow books
- Return books
- Search books by title, author or year
- Generate library report

## Run

python library_system.py

## Concepts

- Object-Oriented Programming
- Classes and Objects
- CLI application
